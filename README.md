# matlib – библиотека для работы с матрицами специальной структуры

**matlib** — это учебный Python-пакет, предназначенный для моделирования и решения линейных задач с матрицами различных форматов.

## Возможности

Пакет ориентирован на **образовательные и исследовательские сценарии**, где важно наглядно представлять структуру матриц, выполнять базовые операции (умножение на вектор, сложение, вычитание, умножение) и решать системы линейных алгебраических уравнений (СЛАУ) с помощью **PLU-разложения**.

В проекте реализованы следующие матрицы:

- **Полные (плотные) матрицы** (`FullMatrix`) – обычные двумерные массивы;

- **Симметричные матрицы** (`SymmetricMatrix`) – экономное хранение только верхнего треугольника;

- **Блочные матрицы** (`BlockMatrix`) – составные матрицы, собранные из произвольных блоков.

## Установка

```bash

pip install matlib-z3342-tem1-msgvk

```

## Быстрый пример

```python

import numpy as np

from matlib_z3342_tem1_msgvk import FullMatrix, solve

A = FullMatrix(np.array([[3.0, 1.0], [1.0, 2.0]]))

b = np.array([9.0, 8.0])

x = solve(A, b)

print(x)  # [2. 3.]

```
